# MicroHAM-MK2
MicroHAM Micro Keyer MK2 Settings for the usage on DX Contest.

## Instalation
From the Router Tab,
* Click Restore Backup Settings (or anything most resembles that)
* Choose the "CQWW - CW - Settings.urs"
* then OK.

The settings shouldbe loaded successfully.
